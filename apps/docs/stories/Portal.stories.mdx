# Portal

The `Portal` component is a React component that provides a way to render children into a DOM node that exists outside the DOM hierarchy of the parent component.

It uses the `ReactDOM.createPortal` API to create a portal and `useEffect` hook to set the target DOM node using the `selector` prop. If the target DOM node is not found, the component will return `null`.

In this example, the Portal component will look for a DOM node with an id of `__portal__`.
