# TexturalVideo

The TexturalVideo component renders a HTML5 video element with primary and secondary video sources. It uses the useIntersectionObserver hook to play/pause the video based on whether it is currently in the viewport. It also has options to set the `className`, `poster` and `threshold` props, and other attributes of the HTML `<video>` element.

## Props

- **`className`** _string_<br />
  The CSS class name to apply to the container element.
- **`primaryVideoUrl`**: _string (required)_<br />
  The URL of the primary video file.
- **`primaryVideoType`**: _string (default: "video/webm")_<br />
  The type of the primary video file.
- **`secondaryVideoUrl`**: _string_<br />
  The URL of the secondary video file.
- **`secondaryVideoType`**: _string (default: "video/mp4")_<br />
  The type of the secondary video file.
- **`poster`**: _string_<br />
  The URL of the poster image for the video.
- **`threshold`**: _number_<br />
  The percentage of the video element that needs to be visible in the viewport to start playing the video.
- Other props: Any other valid HTML `<video>` attributes.

### Basic Usage

```tsx filename="TexturalVideoDemoBasic.tsx"{1, 5-8} copy showLineNumbers
import TexturalVideo from '@madeinhaus/textural-video';

const TexturalVideoDemoBasic: React.FC = () => {
  return (
    <TexturalVideo
      primaryVideoUrl="https://videos.ctfassets.net/j8tkpy1gjhi5/3vr3eDCt4firB4lb5snVrc/c55eb5261efbe9cf0f8489e5aa4b5ab4/Tillamook_Desktop_v7.webm"
      secondaryVideoUrl="https://videos.ctfassets.net/j8tkpy1gjhi5/4YWtNzFROCQBTLFmgt0GtC/46b15d9fbb9a6f5af12819461a37bfc9/Tillamook_Desktop_v7.mp4"
    />
  );
};

export default TexturalVideoDemoBasic;
```

### Transparent Usage

```tsx filename="TexturalVideoDemoTransparent.tsx" {1, 5-9} copy showLineNumbers
import TexturalVideo from '@madeinhaus/textural-video';

const TexturalVideoDemoTransparent: React.FC = () => {
  return (
    <TexturalVideo
      primaryVideoUrl="https://happiest.madeinhaus.com/videos/laughs.mp4"
      primaryVideoType='video/mp4; codecs="hvc1"'
      secondaryVideoUrl="https://happiest.madeinhaus.com/videos/laughs.webm"
      secondaryVideoType='video/webm'
    />
  );
};

export default TexturalVideoDemoTransparent;
```
