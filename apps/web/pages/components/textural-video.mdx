# TexturalVideo

The TexturalVideo component renders a HTML5 video element with primary and secondary video sources. It uses the useIntersectionObserver hook to play/pause the video based on whether it is currently in the viewport. It also has options to set the `className`, `threshold`, `poster`, `title` props, and other attributes of the HTML video element.

## Props

- `className`: A string that represents the class name(s) of the video element. Optional.
- `primaryVideoUrl`: A string that represents the URL of the primary video file. Required.
- `primaryVideoType`: A string that represents the type of the primary video file. Optional, defaults to `video/webm`.
- `secondaryVideoUrl`: A string that represents the URL of the secondary video file. Required.
- `secondaryVideoType`: A string that represents the type of the secondary video file. Optional, defaults to `video/mp4`.
- `poster`: A string that represents the URL of the poster image for the video. Optional.
- `threshold`: A number that represents the percentage of the video element that needs to be visible in the viewport to start playing the video. Optional.
- `title`: A string that represents the title of the video element. Optional.
- Other props: Any other valid HTML video attributes.

### Basic Usage

```tsx filename="TexturalVideoDemoBasic.tsx"{1, 5-8} copy showLineNumbers
import TexturalVideo from '@madeinhaus/textural-video';

const TexturalVideoDemoBasic: React.FC = () => {
  return (
    <TexturalVideo
      primaryVideoUrl="https://videos.ctfassets.net/j8tkpy1gjhi5/3vr3eDCt4firB4lb5snVrc/c55eb5261efbe9cf0f8489e5aa4b5ab4/Tillamook_Desktop_v7.webm"
      secondaryVideoUrl="https://videos.ctfassets.net/j8tkpy1gjhi5/4YWtNzFROCQBTLFmgt0GtC/46b15d9fbb9a6f5af12819461a37bfc9/Tillamook_Desktop_v7.mp4"
    />
  );
};

export default TexturalVideoDemoBasic;
```

### Transparent Usage

```tsx filename="TexturalVideoDemoTransparent.tsx" {1, 5-9} copy showLineNumbers
import TexturalVideo from '@madeinhaus/textural-video';

const TexturalVideoDemoTransparent: React.FC = () => {
  return (
    <TexturalVideo
      primaryVideoUrl="https://happiest.madeinhaus.com/videos/laughs.mp4"
      primaryVideoType='video/mp4; codecs="hvc1"'
      secondaryVideoUrl="https://happiest.madeinhaus.com/videos/laughs.webm"
      secondaryVideoType='video/webm'
    />
  );
};

export default TexturalVideoDemoTransparent;
```
