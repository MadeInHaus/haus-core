# modulo

## Import

```tsx copy
import { modulo } from '@madeinhaus/utils';
```

## Parameters

- `n (number)`: The number to calculate the modulo of.
- `m (number)`: The modulo base.

## Returns

- `(number)`: The modulo of `n` with respect to `m`.

## Usage

```tsx copy showLineNumbers
import { modulo } from '@madeinhaus/utils';

const n = 10;
const m = 3;
const result = modulo(n, m); // 1
```

In this example, we import the `modulo` function from the `@madeinhaus/utils` package and use it to calculate the modulo of a number.
