## Import

```tsx copy
import { last } from '@madeinhaus/utils';
```

## Parameters

- _(array)_: The array to get the last item from.

## Returns

- _(number)_: The last item of the array, or `undefined` if the array is empty.

## Usage

```tsx copy showLineNumbers
import { last } from '@madeinhaus/utils';

const numbers = [1, 2, 3, 4, 5];
const lastNumber = last(numbers); // 5

const emptyArray = [];
const lastItem = last(emptyArray); // undefined
```

In this example, we import the `last` function from the `@madeinhaus/utils` package and use it to get the last item of an array of numbers. We also show that the function returns `undefined` when called with an empty array.
