## Import

```tsx copy
import { preloadImage } from '@madeinhaus/utils';
```

## Parameters

- `url` _(string)_: The URL of the image to preload.

## Returns

- _(Promise\<HTMLImageElement\>)_: A promise that resolves with the preloaded image.

## Usage

```tsx copy showLineNumbers
import { preloadImage } from '@madeinhaus/utils';

async function loadImage() {
  const url = 'https://example.com/image.jpg';
  const image = await preloadImage(url);
  console.log('Image loaded:', image);
}
```

In this example, we import the `preloadImage` function from the `@madeinhaus/utils` package and use it to preload an image from a URL.
