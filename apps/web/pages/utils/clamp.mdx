## Import

```tsx copy
import { clamp } from '@madeinhaus/utils';
```

## Parameters

- `value` _(number)_: The number to clamp.
- `min` _(number)_: The minimum value to clamp to.
- `max` _(number)_: The maximum value to clamp to.

## Returns

- `(number)`: The clamped value.

## Usage

```tsx copy showLineNumbers
import { clamp } from '@madeinhaus/utils';

const value = 10;
const min = 0;
const max = 5;
const clampedValue = clamp(value, min, max); // 5
```

In this example, we import the `clamp` function from the `@madeinhaus/utils` package and use it to clamp a value between a minimum and maximum value.
