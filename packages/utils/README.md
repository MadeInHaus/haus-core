# Utils

**Install**:

```bash
npm i @madeinhaus/utils
```

**Usage**:

```javascript
import { 
    clamp,
    easings,
    getHash,
    getLastSegment,
    last,
    lerp,
    mapRange,
    mappable,
    mergeRefs,
    modulo,
    preloadImage,
    removeHash,
    sign,
    toSlug,
    truncate
} from '@madeinhaus/utils';
```
