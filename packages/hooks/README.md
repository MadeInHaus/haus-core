# Hooks

**Install**:

```bash
npm i @madeinhaus/hooks
```

**Usage**:

```javascript
import { 
    useImagePreload,
    useIntersectionObserver,
    useIsomorphicLayoutEffect,
    usePrevious,
    useRefList,
    useWindowSize,
} from '@madeinhaus/hooks';
```
